# CarlosMenke Tap

## How do I install these formulae?

`brew install carlosmenke/tap/<formula>`

Or `brew tap carlosmenke/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "carlosmenke/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
